# Claude Code 强制开发规则
所有生成、修改的代码必须遵守以下全部约束，不可擅自变更技术栈与规范
## 1. 固定技术栈
- 统一语言：TypeScript
- 前端框架：Next.js
- 后端框架：FastAPI

## 2. 代码编写规范
1. 代码必须模块化拆分，单一文件只承担单一功能
2. 禁止重复代码，通用逻辑抽离公共工具/组件
3. 函数、接口、复杂业务逻辑必须添加清晰注释
4. TypeScript严格类型约束，禁止滥用any

## 3. 部署规范
所有项目配套Dockerfile，支持Docker一键部署

## 4. 文档联动规则
1. 开发前完整读取 /docs 下所有文档
2. 代码修改后同步更新对应文档
3. 新增需求先写入TODO.md，再进行开发
