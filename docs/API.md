# 后端统一接口（FastAPI）
1. POST /api/homework — 创建新作业
2. POST /api/upload/file — 上传标准答案文件
3. POST /api/upload/image — 家长上传作业图片
4. POST /api/ocr — 图片OCR文字识别
5. POST /api/grade — AI自动批改打分
6. GET /api/homework/list — 老师查看全部作业
7. GET /api/submission/list — 获取班级提交记录
8. PUT /api/result/edit — 老师手动修改批改结果
9. GET /api/export/excel — 导出全班成绩Excel
