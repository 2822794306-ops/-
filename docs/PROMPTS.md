# 四大AI提示词固定模板
## Prompt1：标准答案结构化解析
输入：OCR识别文本
输出：标准化JSON，题号+对应答案
{
  "1":"A",
  "2":"12cm",
  "3":"5m/s"
}

## Prompt2：作业自动批改
输入：标准答案JSON、学生作答OCR文本
输出：批改详情JSON，包含总分、每题对错
{
  "score": 95,
  "questions": [
    {"id":1,"correct":true,"reason":"答案匹配"},
    {"id":2,"correct":false,"reason":"单位缺失"}
  ]
}

## Prompt3：班级学情统计分析
输入：全班所有学生批改数据
输出：平均分、整体错误率、高频错题排行文字分析

## Prompt4：家长课后反馈文案
输入：单学生完整批改结果
输出：通俗易懂、发给家长的学情总结文案
