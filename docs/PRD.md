# 第二阶段 MVP 最小可行产品
第一版仅开发6个页面，完整作业批改闭环

## 页面1：老师后台
功能：创建作业

## 页面2：上传标准答案
支持文件类型：PDF、Word、图片

## 页面3：生成上传链接
示例链接：shturl.cc/smcXaUAfEJjhIzKQzYSM

## 页面4：家长上传页面
表单内容：
1. 学生姓名
2. 上传图片
3. 提交按钮

## 页面5：老师查看提交作业
1. 首页展示提交进度（例：32/35）
2. 点击学生查看详情：得分、对错标记、AI批改原因

## 页面6：统计页面
展示数据：
- 平均分
- 错误率
- 错题排行

## 业务流程
1. 老师后台创建作业 → 上传标准答案 → 生成专属上传链接
2. 家长打开链接，填写姓名、上传作业图片并提交
3. AI自动批改作业
4. 老师查看每位学生批改详情，查看全班统计数据
