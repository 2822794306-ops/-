# AI Homework — AI教培教师工作台
项目目标：
降低老师批改作业80%耗时，AI前置自动批改，老师最终审核确认，不替代教师。

## 目标用户
1. 老师（核心使用方）
2. 家长（仅提交作业，无账号体系）
*第一版不单独设计学生账号*

## 老师全部可用功能
✅ 创建作业任务
✅ 上传PDF/Word/图片标准答案
✅ 查看全班家长提交记录
✅ 查看AI自动批改结果
✅ 手动修改AI批改得分与评语
✅ 导出全班成绩Excel报表

## 家长全部可用功能
仅能打开作业专属链接：
填写学生姓名 + 上传作业图片提交，无其他操作。

## AI自动执行全链路流程
OCR文字识别 → 答案结构化解析 → 自动批改打分 → 班级数据统计 → 生成家长可读反馈文案
